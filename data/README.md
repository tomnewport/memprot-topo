# Protein data

Bundled data files for the demo. Run `scripts/download-data.sh` to populate.

## `memprotmd/`

Coarse-grained all-atom PDB files from the [MemProtMD](https://memprotmd.bioch.ox.ac.uk) database.

**Licence:** Creative Commons Attribution 4.0 International (CC BY 4.0)  
**Credit:** Thomas D. Newport, Mark S.P. Sansom and Phillip J. Stansfeld

## `pdbredo/`

DSSP secondary-structure mmCIF files from [PDB-REDO](https://pdb-redo.eu).

**Licence:** Copyright-free; redistribution permitted. Cite PDB-REDO and the original structure authors.

## Underlying PDB entries

All structures originate from the [wwPDB archive](https://www.wwpdb.org), released under CC0 1.0 (public domain).
